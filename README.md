##NYC Taxi Data Engineering Project
This project leverages Databricks to process, analyze, and visualize the New York City Taxi dataset, a large-scale collection of trip records from yellow and green taxis. The goal is to build a robust data engineering pipeline that demonstrates:

Data ingestion: Loading raw taxi trip data from public sources into Databricks.

Data transformation: Cleaning, normalizing, and enriching trip records using Spark.

Data storage: Organizing data into Delta Lake tables for efficient querying.

Analytics & insights: Performing exploratory analysis to uncover trends in ridership, fares, and trip durations.

Scalability: Showcasing how Databricks handles big data workloads with distributed processing.
